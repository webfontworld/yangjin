웹 폰트 사용하기 font-family: 'yangjin';

  • link 방식
  • import 방식
  • CSS 방식
  • 다운로드
  • 라이센스
<link href="https://webfontworld.github.io/yangjin/fonts.css" rel="stylesheet">
@import url('https://webfontworld.github.io/yangjin/fonts.css'); 
@font-face {
    font-family: 'yangjin';
    font-weight: normal; 
    font-style: normal; 
    src: url(https://cdn.jsdelivr.net/gh/webfontworld/Jalnan/yangjin.woff) format('woff');
    font-display: swap;
}
